# Model Exports Directory

This directory holds the serialized model assets generated by `train.py`:
- `best_model.pkl`: The serialized classifier (Logistic Regression or Passive-Aggressive Classifier).
- `vectorizer.pkl`: The serialized TF-IDF vectorizer.
- `model_metadata.pkl`: Training scores, final accuracies, and confusion matrix data.

To generate these files, run `python train.py` from the root directory.
